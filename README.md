# 如何分享你的主题
经测试成功的主题，在 Theme 文件夹下新建你的主题名，将对应的文件放置其中。并且修改索引。
你需要在你的主题文件夹中放置 readMe.txt，描述你的主题理念、修改等等。让用户更好地感受你的想法。
你需要在索引 - 主题介绍中放置一张主题的效果图，让用户预览你的主题效果。
